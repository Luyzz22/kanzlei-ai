- Zweck, Kategorien betroffener Personen, Datenkategorien, Empfänger, Speicherfristen

- Rechtsgrundlage, Datenminimierung, Zugriffskontrollen, Verschlüsselung

- Identifikation, Bewertung, Eintrittswahrscheinlichkeit

## 4. Maßnahmen zur Risikominderung
- RLS, Audit Logging, Retention, Access Controls, Incident Response

## 5. Ergebnis
- Restrisiko + ggf. Konsultation Aufsichtsbehörde
